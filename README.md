# 💍 JEWELUXE Website (Django Project)

An elegant e-commerce website built using Django, designed for selling jewelry online. This site features product listings, categories, image uploads, and admin panel management.

---

## 🌟 Features

- Jewelry product catalog with images and prices
- Admin panel to manage products and categories
- Responsive frontend (optional)
- Future: shopping cart and payment integration

---

## 🛠 Tech Stack

- **Backend**: Django 4.x
- **Frontend**: HTML, CSS (optionally Bootstrap)
- **Database**: SQLite (default for development)
- **Hosting**: Deployable to platforms like Heroku, Vercel, or Render
